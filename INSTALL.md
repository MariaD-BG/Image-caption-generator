## Venv
За да създадете virtual environment, го създавате празен с micromamba или conda и run-вате pip install -e . от папката, в която се намира requirements.txt. Environment-ът трябва да е с python 3.10.

## Dataset
За да свалите данните за трениране, валидация и тестване, отидете на https://www.kaggle.com/datasets/adityajn105/flickr8k и свалете данните за Flickr 8k. Запазете ги в папка data на същото ниво като src
В папката data трябва да има файл captions.txt и папка Images, в която има около 8000 изображения.
